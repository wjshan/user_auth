# 授权码模式

1. 用户打开客户端，客户端将用户导向至认证服务器
2. 用户在认证服务中授权第三方客户端
3. 认证服务重新将用户重定向至客户端指定地址，并附加用户授权码
4. 客服端服务器通过授权码向认证服务器获取access_token与refresh_token

| 参数            | 是否必填 | 说明           |
|---------------|------|--------------|
| response_type | T    | 授权类型 固定为code |
| client_id     | T    | 客户端ID        |
| redirect_uri | F |  重定向url            |
